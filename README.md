# Weather-Dashboard
Weather-Dashboard allow the user to search for weather information by city name. 
By default, the application loads information for the city of Kansas 



# Built with
HTML
CSS
JavaScript
OpenWeather API



# URL
https://s0519.github.io/Weather-Dashboard/
